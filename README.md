A React Native sneaker shopping app built with **Expo**. Browse sneakers, view detailed product information, select sizes, and manage your shopping cart. All in a smooth mobile experience.

---

## Features

- Browse a list of sneakers with images, prices, and sizes
- View detailed product information including multiple images and descriptions
- Add and remove items from your shopping cart
- Select shoe sizes before adding to cart
- Responsive design for mobile screens

---

## Screens

1. **Product Screen** : Displays all available sneakers
2. **Product Details Screen** : Shows detailed info and multiple images of a sneaker
3. **Shopping Cart Screen** : View and manage items you want to purchase

---

## Tech Stack

- React Native  
- Expo  
- JavaScript / JSX  

---

## Installation

1. Clone the repository:  
```bash
git clone https://github.com/yourusername/kickcart.git
```
2. Install dependencies:
```bash
cd kickcart
npm install
```
3. Start the Expo server:
```bash
npm start
```
4. Scan the QR code with the Expo Go app on your phone
(Or run on an emulator using Android Studio or any other mobile development IDE)
